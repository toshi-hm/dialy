## プロジェクト概要

### テーマ
過去の同じ日の記録を見ながら今日の日記をつけられる日記アプリケーション

### コンセプト
"dial" + "diary" = "dialy"

左側上のdialを円状にドラッグして回すことで日付を変更し、過去5年分の同じ日の日記を振り返りながら、今日の日記を記録できる。

### 主要機能
- Dial操作による直感的な日付選択
- 日記の作成・編集・削除
- 過去5年間の同じ日の日記表示
- レスポンシブデザイン対応

### 技術スタック
- Next.js 16
- React 19
- TypeScript 5
- Tailwind CSS 4
- Prisma（ORM - 将来実装）
- Vitest（テスト）
- Storybook（コンポーネント開発）

## 設計方針

### アーキテクチャ原則
- Clean Architecture: ビジネスロジックをフレームワークから独立
- Domain Driven Design: ドメインモデルを中心とした設計
- Atomic Design: UIコンポーネントの階層化

### 開発プロセス
- Test Driven Development: テスト駆動開発
- プロトタイプファースト: UI/UXの早期検証
- 段階的リリース: MVP（最小限の機能）から拡張
